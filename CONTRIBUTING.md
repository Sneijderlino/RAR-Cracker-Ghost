# Contributing to RAR-Cracker-Ghost

Terima kasih atas minat Anda untuk berkontribusi! Berikut adalah cara Anda bisa membantu proyek ini:

1. **Buka issue** jika Anda menemukan bug, kesalahan dokumentasi, atau ingin meminta fitur baru.
2. **Fork** repositori dan buat branch baru:
   ```bash
   git checkout -b nama-fitur-atau-perbaikan
   ```
3. **Lakukan perubahan** di branch baru Anda. Sertakan komentar jelas dan ikuti gaya kode Python yang konsisten.
4. **Tes perubahan Anda** sebanyak mungkin, baik pada Windows maupun Linux bila memungkinkan.
5. **Kirim pull request** dari fork Anda ke repositori utama. Jelaskan apa yang Anda ubah dan alasannya.
6. Setelah PR diterima, Anda dapat menghapus branch dan menarik perubahan terbaru ke salinan lokal.

### Aturan umum

- Gunakan bahasa Inggris untuk nama branch, pesan commit, dan deskripsi PR agar lebih mudah diikuti oleh kontributor internasional.
- Pertahankan antarmuka pengguna tetap sederhana dan ramah pemula.
- Jangan sertakan data pribadi atau password nyata dalam contoh atau test.

Terima kasih atas bantuan Anda!
